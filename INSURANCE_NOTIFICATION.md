## Service Description

### Overview
We have developed a web-based wind loading calculator for signage structures. The tool calculates wind forces on wall-mounted signs in accordance with BS EN 1991-1-4:2005+A1:2010 (UK National Annex) using the methodology documented in SCI Publication P394.

### Service Tiers

#### 1. Free Indicative Calculator (No PI Coverage Required)
- **Purpose:** Lead generation and preliminary design
- **Output:** On-screen wind loading values only
- **Disclaimers:** Heavy disclaimers stating "indicative only", "not for building control"
- **No Certification:** Explicitly states not certified
- **No PDF:** No downloadable reports
- **Liability:** Minimal - users warned values are indicative only

#### 2. Standard PDF Report (£150) - **PI Coverage Requested**
- **Purpose:** Professional documentation
- **Output:** Detailed PDF calculation report
- **Methodology:** Full BS EN 1991-1-4 calculation
- **Certification:** Not engineer-certified
- **Use:** Preliminary design, contractor reference

#### 3. Certified Calculation (£350) - **PI Coverage Requested**
- **Purpose:** Building control submission
- **Output:** Certified PDF report with CEng sign-off
- **Methodology:** Full BS EN 1991-1-4 calculation
- **Certification:** Signed by Toby Fletcher, CEng MIMechE
- **Use:** Building control, planning applications
- **Includes:** Professional indemnity coverage

## Technical Details

### Standard Compliance
- **Primary Standard:** BS EN 1991-1-4:2005+A1:2010
- **UK National Annex:** Fully implemented
- **Reference Document:** SCI Publication P394 "Wind Actions to BS EN 1991-1-4"
- **Validation:** Verified against published worked examples

### Scope of Work
**Current Version (Phase 1):**
- Wall-mounted fascia signs only
- Cantilevered configuration
- Simple rectangular geometry
- UK locations only

**Limitations (Clearly Stated to Users):**
- No orographic effects (hills/cliffs)
- Non-directional wind approach (conservative)
- No complex geometries
- Wind loading only (not structural design of framework/fixings)

### Quality Assurance
1. **Validation:** Calculations validated against SCI P394 Sheffield Bioincubator worked example (Section 9.1)
2. **Peer Review:** To be reviewed by external Chartered Engineer before launch
3. **Testing:** Comprehensive test suite with all tests passing
4. **Documentation:** Full methodology documented with code references
